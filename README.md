# Movie Swipe Club

Небольшой веб‑сервис, где два пользователя подключаются в одну комнату и свайпают фильмы как в Tinder.

## Что умеет
- комната по коду
- подключение двух пользователей
- свайпы вправо/влево
- совпадение, когда оба лайкнули один и тот же фильм
- список из 500 фильмов, собранный из IMDb dataset

## Запуск
```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000` в двух вкладках или на двух устройствах.

## Источник данных
Приложение загружает `title.basics.tsv.gz` и `title.ratings.tsv.gz` из IMDb Non-Commercial Datasets и кэширует результат локально на 24 часа.
